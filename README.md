# Privacy Policy
https://bryncoll.github.io/brync-consulting-g2-api-privacy/

**Last Updated:** June 26, 2026

## Overview

This application is a personal research and software development project used to access the G2 API for data analysis and experimentation.

## Information Collected

If you authorize this application using G2 OAuth, the application may receive information that you explicitly authorize G2 to share, including profile information and API data permitted by the scopes you grant.

The application does not intentionally collect personal information beyond what is required for authentication and authorized API access.

## Use of Information

Information obtained through the G2 API is used solely to:

* Authenticate with G2.
* Retrieve data requested by the user.
* Analyze and research G2 product and category data.
* Develop and test software.

## Data Storage

OAuth access tokens and related credentials may be stored securely for the duration necessary to access the G2 API.

No personal information is sold, rented, or shared with third parties.

## Data Security

Reasonable measures are taken to protect stored credentials and authorized API data from unauthorized access.

## Third-Party Services

This application interacts with G2 through its official API. Use of G2 services remains subject to G2's own privacy policies and terms.

## Changes

This Privacy Policy may be updated periodically. Continued use of the application after changes constitutes acceptance of the revised policy.

## Contact

Questions regarding this Privacy Policy may be directed through the GitHub repository associated with this project.
